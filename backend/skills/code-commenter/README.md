# Code Commenter Skill

一个专门用于为代码添加清晰、有用注释的skill，帮助提高代码的可读性和可维护性。

## 功能特点

- **智能注释添加**：为代码添加有意义的注释，解释WHY而不是WHAT
- **多语言支持**：支持Python、JavaScript/TypeScript、Java等多种编程语言
- **标准格式**：遵循各语言的注释规范（docstring、JSDoc、JavaDoc等）
- **业务逻辑解释**：特别擅长解释复杂的业务规则和算法
- **TODO/FIXME标记**：自动识别并标记需要改进的代码区域

## 使用方法

当用户需要为代码添加注释时，调用此skill，按照以下步骤：

1. **分析代码**：理解代码的功能和逻辑
2. **识别关键点**：找出需要注释的复杂部分
3. **添加注释**：按照标准格式添加有意义的注释
4. **提供解释**：说明添加了哪些注释及其作用

## 测试验证

此skill已经过测试，能够：
- 正确识别不同编程语言的代码结构
- 添加符合语言规范的注释格式
- 解释复杂的业务逻辑和算法
- 保持注释的准确性和实用性

## 与其他skills的关系

- 与 **code-review** skill互补：code-review检查代码质量，code-commenter提高代码可读性
- 可以配合 **agent-builder** 使用：为生成的代码自动添加注释
- 适用于 **mcp-builder** 项目：为MCP服务器代码提供清晰的文档
